Variablen in Javascript ohne Module

Javascript-code der mit normalen script-tags geladen wird befindet sich gemeinsam in einem globalen Namensraum. Hier z.B: Variablen a wird direkt in diesem HTML Dokument gesetzt, und in myscript.js erhöht. Variable b wird in myscript.js gesetzt, und hier im HTML Dokument erhöht.
