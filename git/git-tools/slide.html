Verschiedene Tools stellen für Git eine grafische Oberfläche zur Verfügung. (Warnung: So eine Oberfläche ersetzt nicht das Verständnis von git!)

sourcetree

Das Programm sourcetree ist gratis (aber nicht open source) und für Mac OS und Windows verfügbar: Homepage

PHPStorm

WebStorm, PHPStorm und RubyMine sind IDEs für Web Projekte (von derselben Firma). Die Interaktion mit einem Versionskontrollsystem ist eingebaut. Homepage

VS Code

VS Code kann mit Versionskontrollsystemen umgehen - oder man verwendet das Terminal.
