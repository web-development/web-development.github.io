In unserer Abbildung sehen wir das “lokale Repository” - erst mal als abstrakten Kreis - aber wo ist es “wirklich”? Und wie kann man es erzeugen?

Der geheim Ordner .git

Das lokale Repository befindet sich in einem Ordner, der so geheim ist, dass Windows Explorer und Mac Finder und VS Code und der Kommandozeilenbefehle ls (Mac/UNIX) sie gar nicht anzeigen wollen:

Erst mit der richtigen Anzeige-Einstellung wird der Ordner sichtbar:

git clone

Mit dem Befehl `git clone
