Auf Platz 8 der OWASP Top 10 2017: Unsichere Deserialisierung

Als Serialisierung →wikipedia bezeichnet man in der Informatik die Umwandlung einer Datenstruktur aus dem laufenden Programm in einen String. Ein Beispiel das Sie schon kennen ist das Verwandeln einer JavaScript Arrays in einen String als JSON.

Die Deserialisierung ist dann der umgekehrte Vorgang: die Umwandlung des Strings in eine Datenstruktur.

Der wichtige Tipp für PHP: json_decode und json_encode verwenden.
