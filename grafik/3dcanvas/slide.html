Wenn der canvas Tag in der 3d Variante verwendet wird, kommt wirklich die Grafikkarte mit ihrer ganzen Render-Pipeline zum Einsatz um 3D Szenen zu rendern. Diese API des Browsers wird ‘'’WebGL’’’ genannt, und basiert auf OpenGL ES 2.0.

WebGL wird von allen Browsern unterstützt, sogar am Smartphone.

  • Learning WebGL

Libraries für WebGL

WebGL direkt zu programmieren ist sehr aufwändig. Die Library three.js ermöglicht die direkt arbeit mit 3d-Objekten, Materialien, Szenen, Kameras.
